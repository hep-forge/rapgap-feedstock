[![Build AMD64](https://github.com/hep-forge/rapgap-feedstock/actions/workflows/autoupload.amd64.yml/badge.svg)](https://github.com/hep-forge/rapgap-feedstock/actions/workflows/autoupload.amd64.yml)

Feedstock for [rapgap](https://rapgap.hepforge.org) — part of [hep-forge](https://anaconda.org/hep-forge).

Hadron level Monte Carlo generator for ep and selected processes in pp scattering

## Install

```bash
conda install -c hep-forge -c conda-forge rapgap
```
